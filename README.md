# EKALAVYA-Vision-to-Action-6-DOF-Robotic-Arm
Vision-to-Action 6-DOF Robotic Arm that uses a Raspberry Pi with YOLOv8 (ONNX) to detect and classify objects from a live camera feed, compute target orientation, and command an Arduino-controlled arm for precise pick-and-sort actions. The system performs real-time object sorting (boxes vs cylinders) with live monitoring via a Flask web stream.
